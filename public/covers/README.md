# V2 hero cover photos

Drop **landscape** group/cast photos here. The V2 home page picks them up
automatically and cross-fades through them in the hero cover slot.

## Rules
- Filename: anything ending in `.jpg`, `.jpeg`, `.png`, or `.webp`
- Aspect ratio: roughly landscape (3:2 / 16:9 / 16:10). Tall portraits also
  work but look weird in a landscape slot.
- Recommended size: **at least 1600×1000 px** so it stays sharp on retina.
- Files starting with `_` or `.` are ignored (use `_draft.jpg` to hide a
  WIP shot without deleting it).

## Order
Sorted by filename, so prefix with `01-`, `02-`, etc. to control rotation
order. Example:
```
01-group-rooftop.jpg
02-cast-line-up.webp
03-bts-studio.jpg
```

## Fallback
If this folder is empty, the cover falls back to the 3 featured video
thumbnails set in the admin "이번 호 표지" tab.
